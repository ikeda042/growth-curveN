# 増殖曲線フィッティング

増殖曲線の式を以下のように定義する。

$$f\hat{(t)}=ae^{\mu t} + b$$

この時、亜硝酸消費量と菌体量に線形性がある場合、亜硝酸消費曲線は以下のように表せる。

$$f\hat{(t)}=ae^{-k\mu t} + b$$
